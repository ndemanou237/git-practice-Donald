etape 1.1: lorsque on fait git init dans un repertoire cela cree un depot git , apres avoir taper cette commande un fichier .git est crée et contient les element du projets telle que le head qui a les informations sur la branche que on se trouve actuellement.
etape 1.2: a ce stade git reconnait le fichier readme.md comme un fichier non suivi car bien que le fichier existe dans le repertoire mais le moteur de suivi de git ne le reconnait pas tantque on ne la pas dit de suivre ce fichier et git status permet de voir l'etat acturl de nos fichier depuis notre dernier commit.
etape 1.3:git add permet de faire passer de l'etat d'un fichier de non suivi à suivi tandis que dit commit permet d'enregistrer les modification faite sur les fichiers suivi. git separe ces deux etapes car si on travaille sur plusieurs fichiers on aura l'opportunités de choisir librement les fichiers à enregistrer chaqu'un portant un message clair.
etape 1.4: git log permet de voir l'historique des commits dans un depots, avant un commit la commande git diff ne reagit pas, mais si on avait auparavant faire un commit sur ce depot alors cette commande permettra de voir les mofications que vous avez faite sur les fichiers depuis votre dernier commit.

etape 2.1: un depot local est le travail que l'on a fait sur notre machine alors que un depot distant est notre travaille heberger sur un serveur , git push permet de transferer les données de notre machine vers le serveur distant, si on perd notre machine avant un git push nos données seront completement perdus
etape 2.2: il ne faut jamais versionner le fichier .env car il contient les informations sensibles telque les mot de passe de la base de données , les clés d'api de payement..., ne pas versionner le dossier node_modules car il est lourd et peut etre reinstaller simplement, si ces element sont deja versionner le dossier node_modules sera tres lors lors du push , et apres un git log il aura des fichiers que nous n'avons pas ecrit s'afficher et nos données seront vulnerables car notre fichier .env est exposé.
